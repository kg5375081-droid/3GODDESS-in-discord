# 3GODDESS-in-discord
다야니가 만든 3여신이 디스코드에서 정보를 설명해용...


my_discord_bot.py는 디스코드에서 봇을 구성하는 요소에용..

yukseongma_data_with_aptitude.json은 육성마의 데이터를 불러오는 파일이에용..

-- 명령어 입력 부분이에용--

!육성마 이름 1 
ㄴ 육성마 옷갈유무에 상관없이 기본 육성마의 정보를 출력해용
